# Conversation Handoff

## User Context

The user is a second-year graduate student at Zhejiang University, Software College, Industrial Design Engineering, researching HCI. The immediate pressure is the May proposal defense. Graduation outcomes may include design competition, patent, or paper, but the current priority is to find a convincing thesis topic that can produce a prototype and later support paper/competition output.

## Previous Directions Considered

- Synthetic users for user research: rejected because it felt already crowded and too paper-like.
- Agent vs GUI operation comparison: considered but felt too evaluation-heavy and not distinctive enough.
- AI trust and verifiability for non-experts: considered but not compelling enough.
- Multi-agent orchestration inspired by games/RTS: currently the strongest direction.

## Current Direction

The project now focuses on a formation-based interface for multi-agent collaboration.

Core framing:

> From prompt/chat to formation/command.

The user liked the idea that strategy games contain mature interaction patterns for commanding multiple units. The project should borrow these patterns as interface language, not as superficial gamification.

## Important Tone Preference

The user strongly dislikes ordinary “AI productivity tool” topics. The framing must feel imaginative, future-facing, and prototype-able, while still being defensible for a graduate thesis.
