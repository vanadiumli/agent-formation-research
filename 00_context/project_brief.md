# Project Brief

## Working Title

Chinese: 从对话到编队：面向多智能体协作的指挥式界面研究

English: From Conversation to Formation: Command Interfaces for Human Orchestration of Multi-Agent Collaboration

## Core Question

When AI evolves from a single assistant into a team of agents, what kind of interface helps humans command, understand, and take over collective agent work?

## Core Idea

Current agent interfaces often look like either one conversational assistant or a fixed set of role-based agents. This project explores a different interaction model: the user commands multi-agent collaboration through formations.

A formation is a recognizable collaboration mode for a group of agents, such as scouting, debating, executing, auditing, or handing off. Instead of writing long orchestration prompts or monitoring many independent chat windows, the user switches formations, observes task state, and intervenes when needed.

## Research Framing

This is primarily an HCI and interaction design research project, not an algorithm project. The contribution should focus on the interface model, design vocabulary, prototype, and evaluation of human orchestration over multi-agent systems.

## Likely Contributions

1. A formation-based interaction model for multi-agent collaboration.
2. A catalog of collaboration formations and their interaction states.
3. A prototype interface for switching formations, monitoring agent state, replaying decisions, and intervening.
4. A study plan evaluating whether formations improve user understanding, control, trust calibration, and task coordination.
