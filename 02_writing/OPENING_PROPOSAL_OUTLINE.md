# Opening Proposal Outline

## 1. Research Background

- AI agents are moving from response generation to task execution.
- Multi-agent systems are emerging for complex work.
- Existing interfaces still rely heavily on chat or fixed role panels.
- Users need better ways to understand, command, and intervene in agent collaboration.

## 2. Research Problem

How can users command a team of AI agents without writing complex prompts or monitoring many independent agents?

## 3. Research Objective

Design and evaluate a formation-based interface for multi-agent collaboration.

## 4. Research Questions

1. How can formation-based interaction represent multi-agent collaboration in a way users can understand?
2. Compared with chat-based and fixed-role multi-agent interfaces, does formation-based interaction improve situation awareness and perceived control?
3. How do users decide when to switch formations, intervene, or take over agent work?
4. What design principles can guide future agent command interfaces?

## 5. Research Method

- Literature review.
- Prototype design.
- Controlled comparison study.
- Post-task interviews.
- Qualitative and quantitative analysis.

## 6. Expected Output

- A high-fidelity interactive prototype.
- A user study dataset.
- A thesis/proposal chapter draft.
- A competition-ready interaction design concept.
