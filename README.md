# Agent Formation Research

This is a standalone research project for the thesis/proposal direction:

**从对话到编队：面向多智能体协作的指挥式界面研究**

Working English title:

**From Conversation to Formation: Command Interfaces for Human Orchestration of Multi-Agent Collaboration**

## Current Core Idea

Current AI agent interfaces are often designed as either a single conversational assistant or a fixed group of role-based agents. This project explores a different interaction direction: users manage agent collaboration through **formations**.

A formation is a recognizable collaboration mode for a group of agents, such as scouting, debating, executing, or auditing. Instead of writing long orchestration prompts or monitoring many independent chat windows, users switch formations, observe task state, and intervene when needed.

## Why This Project Exists

The project asks a human-computer interaction question, not an algorithm question:

> When AI evolves from a single assistant into a team of agents, what kind of interface helps humans command, understand, and take over that collective work?
