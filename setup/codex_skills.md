# Codex Skill Setup Notes

## Skills/Plugins Relevant To This Project

### GitHub

Use for repository context, commits, pull requests, and issue tracking.

### skill-creator

Use once the project workflow becomes repeatable. The likely project skill should teach Codex how to help with this research project, including:

- Maintaining the research framing.
- Building and revising a literature map.
- Translating between Chinese and English thesis framing.
- Designing formation catalogs and prototype interaction flows.
- Keeping outputs in the repository structure.

### openai-docs

Use only when implementation requires current OpenAI API or model documentation.

### documents / presentations / spreadsheets

Use later for formal proposal writing, slide decks, or study data analysis.

## Candidate Project-Specific Skill

- "continue agent formation research"

- "draft the proposal intro"
